Paragraph tags have a by default margin of 1 em top and bottom and the proof can be found out in the chrome dev tools by looking at the elements

Block level elements:

Block level elements are those which start on a new line and take the full width available Examples: - p tag , h1-h6,form tags

Inline Elements :

elements or tags do not start on a new line Take only the necessary width Examples : img , link span
